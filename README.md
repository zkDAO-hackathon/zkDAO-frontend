# ZKDAO Frontend

by hackathon

## Quick start 🏄

Prerequisites: [Bun (v1.2.5)](https://bun.sh) and [Git](https://git-scm.com/downloads)

**https://github.com/zkDAO-hackathon**

ZKDAO organization:
```
git clone https://github.com/zkDAO-hackathon/zkDAO-frontend.git
```

> Install dependencies:

```
bun Install
```
